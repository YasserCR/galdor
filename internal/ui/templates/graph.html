Graph viewer

Paste the JSON output of Runnable.Inspect() below and the server renders it as an SVG. Useful for verifying that a graph you built matches the topology you intended.

SVG output appears here.
How to produce a Spec from your code 
import (
    "encoding/json"
    "os"
    "github.com/YasserCR/galdor/pkg/agent"
)

r, _ := agent.NewReAct(agent.Config{ /* ... */ })
_ = json.NewEncoder(os.Stdout).Encode(r.Inspect())
Pipe the result into the textarea above, or POST it to /api/graph/svg to fetch the rendered SVG directly.
